# LangChain-para-JavaScript
LangChain para JavaScript


# DEPENDENCIAS:
npm install @langchain/openai @langchain/langgraph @langchain/core

## @langchain/openai: Integração com os modelos da OpenAI.
## @langchain/langgraph: A biblioteca que permite criar os grafos e fluxos cíclicos.
## @langchain/core: Contém os componentes básicos (mensagens, esquemas, etc.).
